# Run Witch Run — Pure HTML Website

A complete, self-contained website for the Run Witch Run HTML5 endless runner game.

## Pages
- `index.html` — Home page with hero, features, gallery, how-to-play
- `game.html` — Dedicated game page (desktop + mobile optimized)
- `features.html` — All game features
- `gallery.html` — Screenshot gallery with lightbox
- `privacy.html` — Privacy Policy
- `terms.html` — Terms of Service
- `contact.html` — Contact form + FAQ

## Tech Stack
- Pure HTML5, CSS3, JavaScript — zero dependencies
- All images in WebP format
- Game powered by Construct 3 (HTML5/WebGL)

## Deployment
Works on any static host: GitHub Pages, Vercel, Netlify, or any web server.
